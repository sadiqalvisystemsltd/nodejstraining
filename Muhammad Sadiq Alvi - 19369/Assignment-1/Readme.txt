How to run?
From command-line Run "node assignment-1.js"
From browser, hit "http://localhost:3000/read-file", to get TestFile.txt content on browser page.
From browser, hit "http://localhost:3000/read-file-1", to get TestFile1.txt content on browser page.
From browser, hit "http://localhost:3000/read-file-2", to get TestFile2.txt content on browser page.
